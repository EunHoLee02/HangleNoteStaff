# 1단계 PRD

## 서비스 개요

Hangul Note Staff는 한글 음계 텍스트와 텍스트 기반 PDF/TXT 파일을 오선보 이미지 및 MusicXML 악보로 변환하는 웹 서비스다.

## 문제 정의

한글 음계는 교육 현장에서 자주 쓰이지만, 이를 악보 이미지나 MuseScore 같은 악보 프로그램에서 열 수 있는 파일로 변환하려면 별도 지식과 수작업이 필요하다.

## 목표

- P0: `도레미파솔라시도`를 즉시 오선보로 표시한다.
- P0: SVG/PNG/PDF 이미지와 MusicXML을 다운로드하게 한다.
- P0: 긴 악보는 A4 페이지 단위로 미리보기와 다운로드를 제공한다.
- P0: 텍스트 레이어 PDF/TXT 업로드로 음계 입력을 자동화한다.

## 사용자 페르소나

- 34세 음악 학원 강사: 한글 계이름 자료를 악보 이미지로 만들어 수업 자료에 넣고 싶다.
- 15세 초급 학습자: 도레미 입력만으로 오선보 위치를 확인하고 싶다.

## 기능 범위

| 우선순위 | 사용자 기능 |
| --- | --- |
| P0 | 한글 음계 입력, 파싱, A4 오선보 렌더링, SVG/PNG/PDF/MusicXML 다운로드, PDF/TXT 업로드 |
| P1 | OCR PDF, 조표 선택, 박자/마디 자동 보정, 음표 길이 고급 입력 |
| P2 | 계정, 저장, 공유, 악보 히스토리 |

관리자 기능은 MVP에서 불필요하다.

## 비기능 요구사항

- 브라우저에서 즉시 동작해야 한다.
- 네트워크가 없더라도 PDF 추출 외 주요 입력/렌더링은 동작해야 한다.
- 모바일에서는 입력과 악보가 세로 배치되어야 한다.

## 보안/개인정보 요구사항

- P0에서는 서버 업로드 저장을 하지 않는다.
- 파일 내용은 브라우저 메모리에서만 처리한다.
- 민감정보 입력을 요구하지 않는다.

## KPI

- 입력 후 1초 이내 악보 표시.
- SVG/PNG/MusicXML 다운로드 성공률 95% 이상.

## 리스크 및 가정

- 가정: `진짜악보`는 MusicXML 및 표준 오선보 표기를 의미한다.
- 리스크: PDF가 스캔 이미지인 경우 텍스트 추출이 불가능하다.

## 단계 종료 정리

1. 이번 단계 확정 사항: 관리자/인증 제외, 브라우저 정적 MVP, P0 기능 범위.
2. 다음 단계 입력 핵심 요약: 일반 사용자 중심의 한글 음계 변환 기능을 상세 명세로 전개.
3. 추가 확인 필요 항목: OCR을 1차에 포함할지 여부.
4. 충돌 가능 항목: 완전한 PDF 지원과 정적 MVP 사이의 OCR 범위.
