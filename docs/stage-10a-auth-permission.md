# 10a단계 인증 / 권한 구현

## 선행 충돌 점검

- 1~9단계 기준 MVP는 인증, 사용자 저장, 관리자 기능을 제외한다.
- 충돌 항목 없음.

## 구현 범위

인증/권한 기능 없음. 모든 사용자는 비로그인으로 동일 기능을 사용한다.

## 구현 반영

- 서버 세션/JWT 없음.
- 사용자 데이터 저장 없음.
- 파일은 브라우저에서만 처리.

## 테스트 케이스 목록

- 비로그인 상태에서 앱 접속 가능.
- 다운로드 기능이 로그인 없이 동작.
- 파일 업로드가 서버 요청 없이 처리.

## 단계 종료 정리

1. 이번 단계 확정 사항: 인증 미구현이 의도된 P0 정책.
2. 다음 단계 입력 핵심 요약: 백엔드 P0도 없음, 브라우저 모듈로 구현.
3. 추가 확인 필요 항목: 계정 기능 추가 일정.
4. 충돌 가능 항목: 사용자별 악보 저장 요구.
