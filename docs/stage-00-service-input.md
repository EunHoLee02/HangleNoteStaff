# 0단계 서비스 입력 정보 정리

## 1. 서비스 요약

| 항목 | 확정/가정 값 |
| --- | --- |
| 서비스명 | Hangul Note Staff |
| 한 줄 소개 | 한글 음계를 입력하거나 PDF/TXT로 올리면 오선보 이미지와 MusicXML 악보로 변환하는 웹 서비스 |
| 주요 사용자 | 음악 초보자, 어린이 음악 교육자, 한글 음계 자료를 악보로 바꾸는 사용자 |
| 사용 환경 | 웹 |
| 해결 문제 | 한글 `도레미파솔라시도` 텍스트를 실제 오선보로 빠르게 변환하기 어렵다 |
| MVP 필수 기능 | 한글 음계 파싱, 오선보 표시, SVG/PNG/MusicXML 다운로드, 텍스트 PDF 업로드 |
| 1차 출시 제외 기능 | OCR, 로그인, 결제, 다성부 편집, 서버 저장 |
| 파일 업로드 | 필요: PDF, TXT |
| 개인정보/보안 | 계정 없이 브라우저 내 처리, 업로드 파일 서버 저장 없음 |

## 2. 핵심 사용자/문제

- 일반 사용자: 한글 음계를 악보 이미지로 바로 보고 저장하고 싶다.
- 교육자: 수업 자료의 `도레미` 문장을 오선보와 실제 악보 파일로 변환하고 싶다.
- 가정: MVP는 단선율, C major, treble clef 기준으로 시작한다.

## 3. MVP 핵심 범위

- P0: 한글 음계 입력, 붙여쓰기/띄어쓰기 파싱, 오선보 SVG 렌더링, PNG/SVG/MusicXML 다운로드.
- P0: TXT 및 텍스트 레이어 PDF에서 음계 추출.
- P1: 스캔 PDF OCR, 조표/박자 고급 편집.
- P2: 사용자 계정, 악보 저장소, 공유 링크.

## 4. 주요 제약사항

- 브라우저에서 처리하는 정적 웹앱으로 구현한다.
- 스캔 PDF는 OCR 라이브러리 또는 서버 처리가 필요하므로 MVP에서는 제한으로 표시한다.

## 5. 단계 종료 정리

1. 이번 단계 확정 사항: 서비스명, MVP 범위, 파일 업로드 범위, 비저장 원칙.
2. 다음 단계 입력 핵심 요약: 한글 음계를 오선보/SVG/PNG/MusicXML로 변환하는 웹 MVP.
3. 추가 확인 필요 항목: 향후 OCR 필요 여부, 조표 확장 여부.
4. 충돌 가능 항목: “PDF 지원”이 OCR까지 의미하는 경우 MVP 범위와 충돌 가능.
