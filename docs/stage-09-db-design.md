# 9단계 DB 연결 설계

## DB 선택 이유

P0는 악보를 서버에 저장하지 않으므로 DB를 사용하지 않는다.

## 연결 구조 설명

해당 없음. 모든 상태는 브라우저 메모리와 DOM에서 관리한다.

## 환경변수 예시

P0에서 `.env` 없음.

## 마이그레이션/시드 전략

해당 없음.

## 주의사항 및 추가 확인 필요 항목

- 향후 P2 악보 저장 기능이 들어오면 SQLite 또는 PostgreSQL을 검토한다.
- 저장 기능 추가 시 개인정보/저작권/파일 보관 정책을 다시 확정해야 한다.

## 단계 종료 정리

1. 이번 단계 확정 사항: DB 미사용.
2. 다음 단계 입력 핵심 요약: 인증/권한도 P0 미사용.
3. 추가 확인 필요 항목: 저장 기능 출시 여부.
4. 충돌 가능 항목: 저장/공유 기능 요구와 DB 미사용 정책.
