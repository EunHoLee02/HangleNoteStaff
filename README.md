# Hangul Note Staff

한글 음계 `도레미파솔라시도`를 입력하면 A4 오선보 SVG/PNG/PDF와 MusicXML로 변환하는 정적 웹 앱입니다.

## 실행

```powershell
cd C:\Users\eunho\Desktop\VibeCoding\HangulNoteStaff
py -3 -m http.server 5174
```

브라우저에서 `http://localhost:5174`를 엽니다.

## 사용법

- 음계 입력: `도 레 미 파`, `도레미파솔라시도`
- 마디 구분: `|`
- 한 줄 제한: 한 줄에는 최대 4마디만 배치됩니다.
- 도돌이표: `|:` 시작, `:|` 끝, `:|:` 끝/다시 시작, `||` 겹세로줄
- 길이 지정: `/1` 온음표, `/2` 2분음표, `/4` 4분음표, `/8` 8분음표, `/16` 16분음표, `/36` 36분음표
- 쉼표 입력: `쉼`
- 옥타브: `'` 한 옥타브 위, `,` 한 옥타브 아래
- 임시표: `#` 샵, `b` 플랫, `=` 또는 `♮` 제자리표
- 점음표: `.`
- 줄 기호: `~` 타이, `(` 슬러 시작, `)` 슬러 끝
- 아티큘레이션: `>` 악센트, `_` 테누토, `!` 마르카토
- 꾸밈음/장식음: `g` 꾸밈음, `tr` 트릴, `turn` 턴, `mordent` 모르덴트
- 기타 기호: `%` 트레몰로, `f1`~`f5` 운지 번호, `Fine`, `D.C.`, `D.S.`, `Coda`, `Segno` 이동 표식
- PDF/TXT 업로드: 텍스트 레이어가 있는 PDF 또는 TXT에서 한글 음계를 추출합니다.

## 사용 예시

```text
도/1 | 레/2 미/4 파/8 솔/16 | 라/36 시. 도'
```

```text
|: 도 레 미 파 | 솔 라 시 도' | 도' 시 라 솔 | 파 미 레 도 :|
```

```text
도# 레b 미= 파> | 솔_ 라! 시tr 도'g | 도~ 도 레( 미 파)
```

```text
쉼/4 도% 레f2 미Fine || 도레미파솔라시도
```

## 출력물

- `index.html`, `styles.css`, `src/`: 실행 가능한 정적 웹 앱
- `docs/stage-00-service-input.md` ~ `docs/stage-12-test-plan.md`: 프로젝트 산출물
- 내보내기: SVG, PNG, PDF, MusicXML

## 참고

- `/36`은 사용자 요청에 맞춘 앱 내부 기호입니다. MusicXML 표준 note type에는 36분음표가 없어 내보내기에서는 가장 가까운 짧은 꼬리표 계열로 표현합니다.
- MusicXML은 C major, treble clef, 단일 파트 기준의 MVP 형식입니다.
